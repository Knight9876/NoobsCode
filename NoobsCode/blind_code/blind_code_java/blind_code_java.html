- - - - - How To Play - - - - - ______________________________________
1. When you hit the start button, the screen will be divided into two half.

2. Code will be displayed on the right half of the screen.

3. You have to write the code on the left half of the screen.

4. While you are writing the code, you will not be able to see what you are writing since, the writing area will be pitch black.

5. You will have 2 minutes to finish writing the code.
(You can change the timer by clicking on it).

6. Once you are done, click on the submit button to submit your code and to see your results.

7. If you are not able to finish within the given time, once the timer hits
00 : 00, it will automatically submit your code.

Enjoy Yourself.....
